# nyc-real-estate-analysis
NYC real estate transaction analysis using R

## Project structure

```
nyc-real-estate-analysis
├── data
│   ├── NYC_TRANSACTION_DATA.csv
│   ├── NEIGHBORHOOD.csv
│   ├── BOROUGH.csv
│   └── BUILDING_CLASS.csv
├── scripts
│   └── analysis.R
├── plots
│   └── price_distribution.png
└── README.md
```

## Usage

Run the analysis script from the project root:

```bash
Rscript scripts/analysis.R
```
